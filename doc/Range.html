    invert_index() click to toggle source

    Invertiert den Range, mit dem ein Ausschnitt eines Strings oder eines Arrays bestimmt wird. Liefert einen Range, mit dem der inverse Teil des Strings ausgewählt werden kann. Beispiele siehe TestKyaniteRange.

    # File lib/kyanite/range.rb, line 24
def invert_index

  # hinteren Teil ausgeben
  if first == 0
    return (1..0)           if last == -1     # leer
    return (last+1..-1)                       # hinterer Teil

  # vorderen Teil ausgeben
  else 
    return (0..first-1)     if last == -1     # vorderer Teil
    
  end
  
  # äußere Teile ausgeben
  return [(first..-1).invert_index, (0..last).invert_index]
end
