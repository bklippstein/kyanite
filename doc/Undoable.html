    save(method=:dup) click to toggle source

    Speichert ein Objekt. Verwendet standardmäßig Object#dup. Für komplexere Objekte muss allerdings Object#deep_copy genutzt werden.

    # File lib/kyanite/general/undoable.rb, line 18
def save(method=:dup)
  return if self == @@undoable_history[self.object_id] # nichts zu tun
  @@undoable_history[self.object_id] = self.send(method)
end
