    class Array2

    Ein Array2 ist eine spezielle Form von Array, nämlich immer zweidimensional. Siehe auch Matrix2!

    Ein Array2 kann in jeder Zeile eine verschiedene Anzahl Spalten haben (im Gegensatz zu Instanzen der Klasse Matrix2!) Jede Zeile hat einen Zeilenheader (Zelle 0), einen Spaltenheader gibt es nicht.

    # File lib/kyanite/array/array2.rb, line 81
def clean4view(level = 0)
    new = self.collect { |zeile| 
                            if zeile[1].nil? && level >= 1
                                nil
                            elsif zeile[-1] == :more  
                                zeile[0..-2] << zeile[0].dup.more!
                            else
                                zeile
                            end
                          }
    return Array2.new(new.compact)
end

    Gibt das Array2 menschenlesbar aus. Beispielaufruf:

    pp germany.adms_and_orte.inspect
    # File lib/kyanite/array/array2.rb, line 68
def inspect
    analyze = Array.new
    analyze = self.collect do | zeile | 
                                ( zeile.collect { | zelle | zelle.inspect } ).join("  |  ")
                            end # zeile
    return analyze
end
