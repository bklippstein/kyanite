    rephrase_index(i, style=:inv) click to toggle source

    Formuliert den Index eines Arrays

    :pos     als positive Angabe
:neg     als negative Angabe
:inv     wenn positiv, dann negativ und umgekehrt

    Liefert eine Integerzahl.

    Tests and examples see TestKyaniteArray#test_rephrase_index

    # File lib/kyanite/array/array.rb, line 62
def rephrase_index(i, style=:inv)
  return i    if i >  (size-1)
  return i    if i <  -size
  case style
  when :inv
        if i >=0    # pos >> neg
          (i - size)
        else        # neg >> pos
          (i + size)
        end
  when :pos
        if i >= 0   # pos bleibt
          i
        else        # neg >> pos
          (i + size)
        end    
  when :neg
        if i >= 0   # pos >> neg
          (i - size)
        else        # neg bleibt
          i
        end    
  end #case
end
    shift_complement() click to toggle source

    schneidet vorne ein Element ab und gibt den Rest zurück. Wenn der Rest nur noch aus einem Element besteht, wird er ‘pur’ zurückgeliefert und nicht als Array. Nützlich für Rekursionen. Bsp.:

    [1,2,3].shift_complement  =>  [2,3]
[1,2,3].shift_complement.shift_complement  =>  3
[1,2,3].shift_complement.shift_complement.shift_complement  =>  nil

    Tests and examples see TestKyaniteArray#test_array_shift_complement

    # File lib/kyanite/array/array.rb, line 41
def shift_complement
  if self.size > 2
    self[1..-1]
  else
    self[-1]
  end
end
