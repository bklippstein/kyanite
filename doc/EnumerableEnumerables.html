    Enumeration Of Enumerations

    Für zweidimensionale Enumerables bzw. Aufzählungen von Objekten, die wiederum aufzählbar sind. See TestKyaniteEnumerableEnumerables for tests and examples. See ArrayOfEnumerables for an Array with modul EnumerableEnumerables included.

    Macht das Enumerable rechteckig. Maßgeblich ist die erste Zeile.

    Tests and examples see TestKyaniteEnumerableEnumerables.

    # File lib/kyanite/enumerable/enumerable_enumerables.rb, line 28
def rectangle
  qsize = self[0].size   
  result = []
  self.each do |zeile|
    size_diff = qsize - zeile.size  
    # so lassen oder zuschneiden      
    if size_diff <= 0
      result << zeile[0..qsize-1]
    # ergänzen
    else
      result << zeile + ([nil] * size_diff)
    end # if
  end # each zeile   
  result
end
