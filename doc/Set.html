    class Set

    General Set

    Tests and examples see TestKyaniteSet

    Unterschiede der verschiedenen Set-Klassen

    • Ein Set ist ungeordnet.

      Beispiele: TestKyaniteSet#test_set

    • Ein OrderedSet ist geordnet, es behält die ursprüngliche Reihenfolge, wird aber nicht kontinuierlich neu sortiert. Es sei denn, man sorgt dafür mit Dictionary#order_by

      Beispiele: TestKyaniteSet#test_ordered_set

    • Ein SortedSet ist geordnet und sortiert. Es behält immer die Sortierung.

      Beispiele: TestKyaniteSet#test_sorted_set

def last
  first = nil
  @hash.each do |key, value|
    if first.nil?
      first = key
      next
    else
      return key
    end #if
  end #do
